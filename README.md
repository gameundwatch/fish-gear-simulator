# Fish Gear Simulator

釣り具シミュレーター

## Tech Stack

- [SvelteKit](https://kit.svelte.dev/)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [shadcn-svelte](https://next.shadcn-svelte.com/)
- TypeScript
- ESLint / Prettier

## Getting Started

```bash
pnpm install
pnpm dev
```

## Commands

| コマンド | 内容 |
|---|---|
| `pnpm dev` | 開発サーバー起動 |
| `pnpm build` | プロダクションビルド |
| `pnpm preview` | ビルド結果のプレビュー |
| `pnpm check` | 型チェック |
| `pnpm lint` | ESLint |
| `pnpm format` | Prettier |
